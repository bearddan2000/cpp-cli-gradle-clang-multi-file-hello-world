# cpp-cli-gradle-clang-multi-file-hello-world

## Description
This is a POC project to demonstrate a
gradle as a build tool for c++.

## Tech stack
- clang
- gradle
- c++

## Docker stack
- gradle:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
